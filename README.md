# Anthony Lall Portfolio

A simple static portfolio site built with plain HTML, CSS, and JavaScript.

## Structure

- `index.html` - main page
- `main/styles.css` - site styling
- `main/script.js` - small interactive behavior

## Run

Open `index.html` directly in a browser, or serve the folder with any static file server.

No build step, framework, or package install is required.
